== Description ==

shows in your post a list of related videos.

== Installation ==


1. Upload `Related Video` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3 Add custom fields in your post:

Custom Field: tags_youtube
Action:Tags to search (Only One)
Values: Null, or Only One Tag
Example: Nasa

Custom Field: related_video
Action: Active or Desactive
Values: Null, 0, 1
0=> Desactive plugin in the post
1 and Null => Active plugin in the post

Custom Field:title_youtube
Action: Show title in the list
Example: Related Videos
